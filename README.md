# 团体邮轮旅行方案 · Claude Code 插件

为公司团体或组织提供邮轮旅行方案，适合团队出行的需求，制定专属的邮轮产品。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/craftwave-skill-6-claude-plugin
/plugin install craftwave-skill-6@craftwave-skill-6-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **团体邮轮旅行方案**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
